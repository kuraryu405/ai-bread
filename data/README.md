# 学習データ

このリポジトリは、課題確認時にそのまま学習できるように画像データを同梱しています。

`npm run organize-data` を実行すると、ユーザーが保管している元写真から次のディレクトリへ**コピー**し直します。
元画像は変更・移動しません。

- `training/bread`: ブレッド 31 枚
- `training/baguette`: バゲット 36 枚
- `training/danish`: デニッシュ 48 枚
- `training/roll`: ロール 37 枚
- `training/twist`: ツイスト 35 枚
- `training/campagne`: カンパーニュ 34 枚（HEIC から JPEG へ変換）
- `evaluation/mixed`: 複数のパンが写る確認用画像 6 枚
